Java
Java とは
・Javaは、汎用プログラミング言語とソフトウェアプラットフォームの
・双方を指している総称ブランドである。オラクルおよびその関連会社の登録商標である。
・1996年にサン・マイクロシステムズによって市場リリースされ、
   2010年に同社がオラクルに吸収合併された事によりJavaの版権もそちらに移行した。
メッセージ表示
・System.out.println("表示文字") で、メッセージを出力可能。
・また、public static void main(String[] args) は、魔法のおまじない。
コメント
・ソースコードにメモを残す際に活用する。
・コンピュータ側からは、読み込まれない。
計算
・四則演算も容易。
・余りの計算も可能。
変数
・java の場合データ型の指定に気を付ける。
・文字列と数値で異なる。
