HTML/CSS
HTML とは
・HTMLまたはHyperText Markup Languageは、ハイパーテキストを記述するためのマークアップ言語の1つで、プログラミング言語ではない。
・主にWorld Wide Webにおいて、ウェブページを表現するために用いられる。
・実際にこのWebサイトにも、HTMLが組み込まれている。
CSS とは
・Cascading Style Sheetsは、HTML や XML の要素をどのように修飾するかを指示する仕様の一つで、
   World Wide Web Consortium がとりまとめ勧告する、文書の構造と体裁を分離させるという
   理念を実現するために提唱されたスタイルシートの具体的な仕様の一つ。
